## 1. Общие понятия
- Рациональная дробь — отношение многочленов $\frac{P(x)}{Q(x)}$.
- Правильная дробь: степень $P$ < степени $Q$. Неправильная: степень $P$ ≥ степени $Q$ (сначала выделяют целую часть делением).
- Любую правильную дробь можно разложить в сумму **простейших дробей** четырёх типов.
## 2. Четыре типа простейших дробей
### Тип 1
- Вид: $\frac{A}{x-a}$, где $A,a\in\mathbb{R}$.
- Интеграл: $$\int \frac{A}{x-a}dx = A\ln|x-a| + C$$.
### Тип 2
- Вид: $$\frac{A}{(x-a)^k}$$, где $k\ge 2$, $A,a\in\mathbb{R}$.
- Интеграл: $$\int \frac{A}{(x-a)^k}dx = A\cdot\frac{(x-a)^{-k+1}}{-k+1} + C = \frac{A}{(1-k)(x-a)^{k-1}} + C$$.
### Тип 3
- Вид: $$\frac{Ax+B}{x^2+px+q}$$, где $p^2-4q<0$ (квадратный трёхчлен не разлагается на вещественные линейные множители).
- **Метод интегрирования:**
  1. Выделить полный квадрат в знаменателе: $$x^2+px+q = (x+\frac{p}{2})^2 + a^2$$, где $a = \sqrt{q-\frac{p^2}{4}} > 0$.
  2. Представить числитель: $Ax+B = A(x+\frac{p}{2}) + \left(B - \frac{Ap}{2}\right)$.
  3. Разбить интеграл на два:
     - $$\int \frac{A(x+\frac{p}{2})}{(x+\frac{p}{2})^2+a^2}dx = \frac{A}{2}\ln\left((x+\frac{p}{2})^2+a^2\right) + C$$.
     - $$\int \frac{B - \frac{Ap}{2}}{(x+\frac{p}{2})^2+a^2}dx = \frac{B - \frac{Ap}{2}}{a}\,\arctan\frac{x+\frac{p}{2}}{a} + C$$.
- **Итоговая формула:**
  $$\int \frac{Ax+B}{x^2+px+q}dx = \frac{A}{2}\ln(x^2+px+q) + \frac{2B-Ap}{\sqrt{4q-p^2}}\,\arctan\frac{2x+p}{\sqrt{4q-p^2}} + C$$.
### Тип 4
- Вид: $$\frac{Ax+B}{(x^2+px+q)^n}$$, где $n\ge 2$, $p^2-4q<0$.
- **Метод интегрирования:**
  1. Выделить полный квадрат в знаменателе: $(x+h)^2 + a^2$.
  2. Разбить на два интеграла (аналогично типу 3):
     - $$\int \frac{A(x+h)}{((x+h)^2+a^2)^n}dx$$ — замена $t=(x+h)^2+a^2$, $dt=2(x+h)dx$,
	результат $$\frac{A}{2}\int t^{-n}dt = \frac{A}{2(1-n)}t^{1-n}+C$$.
     - $$\int \frac{const}{((x+h)^2+a^2)^n}dx$$ — вычисляется по **рекуррентной формуле**:
       $$I_n = \int \frac{dt}{(t^2+a^2)^n} = \frac{t}{2a^2(n-1)(t^2+a^2)^{n-1}} + \frac{2n-3}{2a^2(n-1)} I_{n-1}$$
       где $I_1 = \frac{1}{a}\arctan\frac{t}{a}+C$.
## 3. Алгоритм интегрирования любой рациональной дроби
1. Если дробь неправильная — разделить числитель на знаменатель, получить целую часть (многочлен) и остаток (правильную дробь).
2. Знаменатель правильной дроби разложить на линейные и квадратичные множители.
3. Записать разложение на простейшие дроби с неопределёнными коэффициентами (метод неопределённых коэффициентов).
4. Найти коэффициенты (приравниванием числителей или подстановкой корней).
5. Проинтегрировать каждую простейшую дробь по типам 1–4.
6. Сложить результаты (интеграл от многочлена даётся таблично).
## 4. Пример
$$\int \frac{x+1}{x^2-5x+6}dx$$
- Знаменатель: $x^2-5x+6 = (x-2)(x-3)$.
- Разложение: $\frac{x+1}{(x-2)(x-3)} = \frac{A}{x-2} + \frac{B}{x-3}$.
- Находим $A,B$: $x+1 = A(x-3)+B(x-2)$.
  При $x=2$: $3 = A(-1) \Rightarrow A=-3$.
  При $x=3$: $4 = B(1) \Rightarrow B=4$.
- Интеграл: $$\int \frac{-3}{x-2}dx + \int \frac{4}{x-3}dx = -3\ln|x-2| + 4\ln|x-3| + C = \ln\left|\frac{(x-3)^4}{(x-2)^3}\right| + C$$.
## 5. Итог
- Простейшие дроби типов 1–2 дают логарифмы и степенные функции.
- Тип 3 даёт логарифм и арктангенс.
- Тип 4 сводится к рекуррентному соотношению.
- Метод универсален для любых рациональных дробей.
