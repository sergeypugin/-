## 1. Формулировка теоремы

**Теорема (о разложении правильной рациональной дроби на простейшие).**  
Пусть $P(x)/Q(x)$ — правильная рациональная дробь, т.е. $\deg P < \deg Q$.  
Знаменатель $Q(x)$ разложен на множители над $\mathbb{R}$:  
$$Q(x) = (x - a_1)^{k_1} \cdots (x - a_r)^{k_r} \cdot (x^2 + p_1 x + q_1)^{\ell_1} \cdots (x^2 + p_s x + q_s)^{\ell_s},$$ где $p_j^2 - 4q_j < 0$.  
Тогда $P(x)/Q(x)$ единственным образом представляется в виде суммы простейших дробей:  
$$\frac{P(x)}{Q(x)} = \sum_{i=1}^{r}\sum_{j=1}^{k_i}\frac{A_{ij}}{(x-a_i)^j} + \sum_{u=1}^{s}\sum_{v=1}^{\ell_u}\frac{B_{uv}x+C_{uv}}{(x^2+p_u x+q_u)^v},$$ где $A_{ij},B_{uv},C_{uv}\in\mathbb{R}$.

## 2. Леммы

**Лемма 1 (линейный множитель).** Если $a$ — корень $Q(x)$ кратности $k\ge1$, то существуют число $A$ и правильная дробь $P_1/Q_1$ ($Q_1=Q/(x-a)$) такие, что $$\frac{P}{Q} = \frac{A}{(x-a)^k} + \frac{P_1}{Q_1}.$$

**Лемма 2 (квадратичный множитель).** Если $x^2+px+q$ — неприводимый множитель $Q(x)$ кратности $\ell\ge1$, то существуют $B,C$ и правильная дробь $P_2/Q_2$ ($Q_2=Q/(x^2+px+q)$) такие, что $$\frac{P}{Q} = \frac{Bx+C}{(x^2+px+q)^\ell} + \frac{P_2}{Q_2}.$$

## 3. Доказательство (индукция по $\deg Q$)

**База:** $\deg Q=1$, $Q=x-a$, дробь $P/(x-a)$ уже простейшая.

**Шаг:** пусть верно для всех дробей со знаменателем степени $<n$, $\deg Q=n$.

- Случай 1 (действительный корень $a$ кратности $k$): по лемме 1, $P/Q = A/(x-a)^k + P_1/Q_1$, где $\deg Q_1 = n-k < n$. По индукции $P_1/Q_1$ разлагается на простейшие. Добавляя $A/(x-a)^k$, получаем разложение. Единственность — из однозначности $A$ и индукции.

- Случай 2 (нет действительных корней, есть квадратичный множитель $x^2+px+q$ кратности $\ell$): по лемме 2, $P/Q = (Bx+C)/(x^2+px+q)^\ell + P_2/Q_2$, где $\deg Q_2 < \deg Q$. По индукции $P_2/Q_2$ разлагается, добавляем $(Bx+C)/(...)^\ell$.

## 4. Следствия

- Любая рациональная дробь интегрируется в элементарных функциях.
- Метод неопределённых коэффициентов обоснован.
- Разложение существует и единственно.

## 5. Пример

Разложить $(x+1)/(x^2-5x+6)$.  
$$x^2-5x+6=(x-2)(x-3)$$ ⇒ $$(x+1)/((x-2)(x-3)) = A/(x-2)+B/(x-3)$$.  
$x+1 = A(x-3)+B(x-2)$. 
Подстановка $x=2$: $3=-A\Rightarrow A=-3$; $x=3$: $4=B\Rightarrow B=4$.  
Ответ: $\frac{-3}{x-2}+\frac{4}{x-3}$.

## 6. Заключение

Теорема — основа интегрирования рациональных функций.
