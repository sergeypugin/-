## Многочлены Бернштейна

### Определение

Пусть $f(x)$ — функция, непрерывная на отрезке $[0, 1]$. Многочленом Бернштейна степени $n$ для функции $f$ называется выражение вида:

$$B_n(f; x) = \sum_{k=0}^{n} f\left(\frac{k}{n}\right) p_{n,k}(x), \quad \text{где } p_{n,k}(x) = C_n^k x^k (1-x)^{n-k}$$

Здесь $C_n^k = \frac{n!}{k!(n-k)!}$ — биномиальные коэффициенты.
### Вероятностная интерпретация
Рассмотрим схему из $n$ независимых испытаний Бернулли с вероятностью успеха $x \in \langle 0, 1 \rangle$. Пусть $S_n$ — случайная величина, равная количеству успехов. Вероятность того, что произойдет ровно $k$ успехов, задается формулой:

$$P(S_n = k) = C_n^k x^k (1-x)^{n-k}$$

Известно, что для данной случайной величины математическое ожидание $E[S_n] = nx$, а дисперсия $D(S_n) = nx(1-x)$.

### 1. Первое тождество

$$\sum_{k=0}^{n} C_n^k x^k (1-x)^{n-k} = 1$$

#### Доказательство

- Данная сумма представляет собой полную вероятность всех возможных исходов в схеме из $n$ испытаний:
    
    $$ \sum_{k=0}^{n} P(S_n = k) = 1$$
    
- Согласно формуле бинома Ньютона, это выражение тождественно равно $(x + (1-x))^n = 1^n = 1$.
    

### 2. Второе тождество

$$\sum_{k=0}^{n} \frac{k}{n} C_n^k x^k (1-x)^{n-k} = x$$

#### Доказательство

- Выражение в левой части можно записать через математическое ожидание случайной величины $S_n$:
    
    $$ \sum_{k=0}^{n} \frac{k}{n} P(S_n = k) = \frac{1}{n} E[S_n]$$
    
- Так как $E[S_n] = nx$, получаем:
    
    $$ \frac{1}{n} (nx) = x$$
    

### 3. Третье тождество

$$\sum_{k=0}^{n} \frac{k^2}{n^2} C_n^k x^k (1-x)^{n-k} = x^2 + \frac{x(1-x)}{n}$$

#### Доказательство

- Левая часть есть математическое ожидание квадрата величины $S_n/n$:
    
    $$ \sum_{k=0}^{n} \frac{k^2}{n^2} P(S_n = k) = E\left[ \left( \frac{S_n}{n} \right)^2 \right] = \frac{1}{n^2} E[S_n^2]$$
    
- Используя связь момента второго порядка с дисперсией $E[S_n^2] = D(S_n) + (E[S_n])^2$:
    
    $$ E[S_n^2] = nx(1-x) + (nx)^2 = nx(1-x) + n^2x^2$$
    
- Подставляя это в выражение:
    
    $$ \frac{nx(1-x) + n^2x^2}{n^2} = \frac{nx(1-x)}{n^2} + \frac{n^2x^2}{n^2} = \frac{x(1-x)}{n} + x^2$$
    
    Что и требовалось доказать.
