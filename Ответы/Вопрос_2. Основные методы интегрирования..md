## 1. Метод замены переменной (подстановки)
**Идея.** 
Упростить интеграл, введя новую переменную $t = \varphi(x)$, чтобы получить табличный интеграл.
**Формула.**
$\int f(\varphi(x))\,\varphi'(x)\,dx = \int f(t)\,dt$, где $t=\varphi(x)$, $dt=\varphi'(x)dx$.
**Порядок действий.** 
1) Выбрать замену $t = \varphi(x)$. 
2) Найти дифференциал $dt = \varphi'(x)dx$. 
3) Переписать интеграл в переменной $t$. 
4) Вычислить табличный интеграл. 
5) Вернуться к $x$.
**Пример.**
$\int 2x\cos(x^2)dx$. 
Замена $t=x^2$, $dt=2xdx$. 
Получаем $$\int \cos t\,dt = \sin t + C = \sin(x^2)+C$$.
## 2. Метод интегрирования по частям
**Идея.** Преобразовать интеграл от произведения функций, используя формулу дифференциала произведения.
**Формула.** $\int u\,dv = uv - \int v\,du$, где $u=u(x)$, $dv=v'(x)dx$ (выбираются так, чтобы новый интеграл $\int v\,du$ был проще исходного).
**Правило выбора $u$ и $dv$.** За $u$ обычно берут функцию, которая упрощается при дифференцировании (логарифм, аркфункция, многочлен), а за $dv$ – оставшуюся часть с $dx$.
**Пример 1.** 
$\int x e^x dx$. Пусть $u=x$, $dv=e^x dx$. Тогда $du=dx$, $v=e^x$. По формуле: $$x e^x - \int e^x dx = x e^x - e^x + C = e^x(x-1)+C$$.
**Пример 2.**
$\int \ln x\,dx$. 
Пусть $u=\ln x$, $dv=dx$. 
Тогда $du=\frac{1}{x}dx$, $v=x$. Получаем: $$x\ln x - \int x\cdot\frac{1}{x}dx = x\ln x - \int 1\,dx = x\ln x - x + C$$.
**Пример 3.** 
$\int x\sin x\,dx$. $u=x$, 
$dv=\sin x dx$ $\Rightarrow$ $du=dx$, $v=-\cos x$. 
Результат: $$-x\cos x + \int \cos x\,dx = -x\cos x + \sin x + C$$.
## 3. Дополнительные замечания
- Замена переменной и интегрирование по частям – два основных технических приёма.
- Иногда их применяют последовательно (например, сначала замена, затем по частям).
- Для рациональных дробей также используют метод разложения на простейшие дроби, но это частный случай.
**Итог.** Методы позволяют свести сложные интегралы к табличным. Без них вычисление большинства неопределённых интегралов невозможно.
